# Code of Conduct

## Naming convention
Functions: PascalCase
    - Methods: camelCase

Variables: camelCase
    - member variables: Prefixed with "m_" (Part of Hungarian notation)
    - static variables: PascalCase

Classes & Structs: PascalCase

Enums & Constants: ALL_CAPS with an underscore seperating words
